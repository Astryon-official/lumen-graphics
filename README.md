# Lumen

**Light every world.**

Lumen is a modern graphics enhancement framework by Astryon designed to bring advanced rendering techniques, lighting, and visual effects to games and desktop environments.

> ⚠️ Lumen is currently in early development.

## Vision

Lumen aims to provide a high-performance rendering framework that can be adapted to multiple games and applications.

Planned goals include:

- Dynamic lighting
- HDR rendering
- Bloom
- Shadow mapping
- Screen-space reflections
- Ambient occlusion
- Global illumination
- Plugin architecture
- Multi-game support

## Planned Platforms

- Geometry Dash
- Minecraft
- Desktop enhancements
- Additional games (future)

## Philosophy

Lumen is designed to be:

- Fast
- Modern
- Modular
- Extensible
- Cross-platform
